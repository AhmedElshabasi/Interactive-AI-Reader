# Interactive AI Reader Desktop (Electron)

This project can now run as a desktop app with local Piper TTS (no localhost backend required for speech).

## 1) Install frontend dependencies

From `pdfreader-frontend/`:

```bash
npm install
```

## 2) Add Piper assets

Copy these files into `pdfreader-frontend/local-tts/`:

- `piper.exe`
- `en_US-ryan-high.onnx`
- `en_US-ryan-high.onnx.json` (if required by the voice)

## 3) Run desktop app in development

```bash
npm run electron:dev
```

This starts React and opens Electron with IPC-based local TTS.

## 4) Build Windows installer/exe

```bash
npm run electron:dist
```

Output is generated by `electron-builder` under `pdfreader-frontend/dist/`.

## Notes

- In desktop mode, TTS uses local `piper.exe` through Electron IPC.
- If local assets are not found, you can override paths with:
  - `PIPER_BINARY_PATH`
  - `PIPER_MODEL_PATH`
- ChatGPT text cleanup still uses `REACT_APP_API_BASE_URL` if configured; otherwise the app falls back to raw extracted text.
